# Panel

## Description

The Panel widget wraps other widgets in a consistent panel design.

## Options

| Attribute  | Type     | Description
| -          | -        | -
| `header?`  | `string` | Displays text before the wrapped element
| `footer?`  | `string` | Displays text after the wrapped element

## Code example

```html
<ais-panel
  header="Header foo"
  footer="Footer bar"
>
</ais-panel>
```
